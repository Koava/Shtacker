CreateJS Module

Defined in: CreateJS:45

A collection of Classes that are shared across all the CreateJS libraries. The classes are included in the minified files of each library and are available on the createsjs namespace directly.

Example

myObject.addEventListener("change", createjs.proxy(myMethod, scope));

This module provides the following classes:

  • Event
  • EventDispatcher
